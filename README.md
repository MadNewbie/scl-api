1. pastikan python 3 terpasang pada komputer
2. install pip
3. install package venv
4. jika venv sudah terpasang, buat virtual environment(venv) pada folder sistem dengan menjalankan perintah "python -m venv ."
5. aktifkan venv dengan menjalankan perintah "./Scripts/activate"
6. install requirement system yang tertera pada file requirement.txt dengan menjalankan perintah "pip install -r requirement.txt"
7. system sudah siap dijalankan dengan perintah "fastapi dev"
